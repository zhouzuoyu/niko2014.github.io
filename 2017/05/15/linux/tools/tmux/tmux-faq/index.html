远程登录时, 显示空间很小

方法一:

取消所有 client 的 attach 状态
tmux detach -a
tmux ls , 检查是否都 detach 了, 如果还不行, 那么
ps -ef | grep tmux, 找出还在 attach 某个 session 的 client, kill 之.

方法二 (未验证) :

使用以下设置, 那些点点就会被作为可用空间了. 可能是不同 client 的显示器不同导致.
set-window-option -g aggressive-resize
