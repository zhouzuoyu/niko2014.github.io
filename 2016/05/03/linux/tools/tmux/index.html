tmux cheatsheet

发表于 2016-05-03   |   分类于 linux   |  

tmux是一个终端的terminal multiplexer, 用来切换多个terminal, 和之前博客提到的terminator比较, 可以发现, tmux更适合在纯文字终端中使用(比如ssh登录远程服务器), 还具有session等强大的功能特性.

tmux session

使用tmux, 需要打开一个session, 只要这个session没有被关闭, 下次可以重新使用(attach). 比如远程连接断开, 下次再连上服务器时, 原先tmux打开的环境都仍保留着.

  • start new session:
    tmux new -s myname

  • list sessions:
    tmux ls

  • attach:
    tmux a

  • attach to named:
    tmux a -t myname

  • detach
    ctrl + b, d

pane

以下命令都需要先按下ctrl + b:

快捷键 说明
% 垂直分割
“ 水平分割
o panes切换:
x 关闭当前pane
q 显示 pane number
空格 切换不同的layout

window

以下命令都需要先按下ctrl + b:

快捷键 说明
c 新建窗口
n 切换下一个窗口
p 切换前一个窗口
Shift+7 关闭当前窗口
, 命名窗口
. 移动窗口 (使用序号)
w list window
& kill window
page-up 进入滚动模式, 按esc退出滚动

window/pane间操作

以下命令都需要先按下ctrl + b:

快捷键 说明
:joinp -s :2 移动窗口2作为当前窗口的一个新pane
:joinp -t :1 移动当前pane作为窗口1的一个新pane
